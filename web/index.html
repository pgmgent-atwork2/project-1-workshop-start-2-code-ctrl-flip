Mappenstructuur

Download hier het start structuur. Je vind hierin 4 mappen: een CSS, een data een fonts en een images folder, en een "index.html" bestand.

  2. Voeg een nieuwe folder toe genaamd "script". Zorg ervoor dat deze folder op dezelfde plaats staat als de andere folders.
